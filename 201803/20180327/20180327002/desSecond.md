# HTML
所有未知的缩写都会转换成标签，例如，foo → <foo></foo>
## 缩写：!
```html
<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>

</body>
</html>
```
