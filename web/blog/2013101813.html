VMware网络连接方式

Bridged:Connected directly to the physical network
NAT:Used to share the host's IP address
Host-only:A private network shared with the host
Custom:Specific virtual network
VMware有两块虚拟网卡分别是：VMware Virtual Ethernet Adapter for VMnet1和VMware Virtual Ethernet Adapter for VMnet8。
网络连接设置为NAT的时候，使用的是虚拟网卡VMnet1，虚拟机分配的ip地址是192.168.136.128，主机的IP地址为192.168.136.1，虚拟机和主机之间可以共享文件夹，可以进行远程访问和设置服务器客户机。
网络连接设置为NAT的时候，使用的是虚拟网卡VMnet8，虚拟机分配的ip地址是192.168.88.128，主机的IP地址为192.168.88.1，虚拟机和主机之间可以共享文件夹，可以进行远程访问和设置服务器客户机。
